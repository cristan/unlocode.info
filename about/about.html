UN/LOCODE: About us

This website is basically the UN/LOCODE dataset in website form. The dataset actually used is https://github.com/datasets/un-locode as it's a more handy release (for example: special characters actually work there).

Annoyed about the missing and incorrect coordinates? Check out github.com/cristan/improved-un-locodes for a dataset with much improved coordinates. You can get into touch with me here if you want to. I didn't use this dataset on this website though, as I wanted this site to reflect the official dataset.

See a mistake in the UN/LOCODE data? Do a maintenance request right at the source! A new dataset is released once or twice per year, so it will take a while before any changes will show up.
