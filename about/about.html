UN/LOCODE: About us

This website is basically the UN/LOCODE dataset in website form. The dataset actually used is https://github.com/datasets/un-locode as it's a more handy release (for example: special characters actually work there).

This website is open source! See github.com/cristan/unlocode.info for its source. Any issues/suggestions for improvements can be reported here. PR's welcome!

As you explore the site, you'll notice many missing and incorrect coordinates. If this is an issue, check out github.com/cristan/improved-un-locodes. It has dramatically better coordinates. This dataset isn't used on this website though, as I wanted this site to reflect the official dataset.

Want to see a change in the UN/LOCODE data? Do a maintenance request right at the source! A new dataset is released once or twice per year, so it will take a while before any changes will show up.
